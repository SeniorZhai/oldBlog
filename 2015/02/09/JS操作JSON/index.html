JSON本身是JavaScript原生格式，这意味着JS操作JSON起来比其他语言都要优雅。

JSON的结构一般分为对象和数组
对象以{和}包括，例如{'a':'a','b':1234,c:'2008-08-08'}
数组以[和]包括，例如[{'a':'a','b':1234,c:'2008-08-08'},{'a':'a','b':1234,c:'2008-08-08'}]
