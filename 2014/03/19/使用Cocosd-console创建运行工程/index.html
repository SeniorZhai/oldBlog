##运行环境

  • Android2.3及以上版本
  • iOS 5.0及以上版本
  • OS X 10.7及以上版本
  • Windows 7及以上版本
  • Ubuntu 12.04及以上版本
  • Cocos2d-X v3.0rc

##软件需求

  • Xcode 4.6
  • gcc 4.7
  • Visual Studio 2012
  • Python 2.7.5

##创建新工程
切换到cocos的根目录

  • MyGame:指定工程名
  • - p com.zoe.MyGame:指定Android的包名
  • -l cpp:指定项目的语言，可用cpp或者lua
  • -d ./MyProject:指定存放目录
