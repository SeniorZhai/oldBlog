对于广大码农来说，翻墙的主要目的就是Google一下，广大的Android Dev们更是需要SDK(为什么天朝连SDK都不让下)，下面推介一个简单易用的好办法。

##Windows

  1. 请先退出所有杀毒软件
  2. 使用管理员身份运行
  3. 使用及更新: fuckGFW-64.exe(64位) / fuckGFW-32.exe(32位)双击运行
    ##*nix/OS X
    在终端运行$ curl http://tx.txthinking.com/fuckGFW.py | sudo python
    ##使用注意点
  • 防止google本地化重定向: 访问https://www.google.com/ncr
  • 请使用https替代http访问
