CocoaPods是一个objc的依赖管理工具，本身利用ruby的依赖管理gem进行构建

##下载及安装
本地安装好Ruby环境，在Terminator中输入以下命令

sudo gem install cocoapods

由于cocoapods.org会被墙，所以需要使用淘宝镜像来访问

##使用CocoaPods
示例导入AFNetworking
